Blog Header Image Generator — Coding Challenge

Workflow

Blog-Titel von endometriose.app → Prompt Engineering → DALL-E 3 (HD) → Brand-Overlay (Pillow) → Fertiges Header-Bild

Jedes Bild wird mit einem konsistenten Style-Prefix generiert (Farbpalette, Stil, Komposition) und anschliessend mit Comfortaa-Titel und Teal-Gradient nachbearbeitet. Die Blog-Titel stammen direkt von endometriose.app.

Wie wird Konsistenz erreicht?

1. Style-Prefix Identische Stilbeschreibung (Farben, Stimmung, Komposition) als Prefix in jedem DALL-E 3 Prompt.
2. Titelspezifische Szene Jeder Blog-Titel bekommt eine eigene visuelle Beschreibung, die zum Thema passt.
3. Brand-Overlay Pillow fuegt Teal-Gradient + Comfortaa-Titel hinzu — einheitliches Branding auf jedem Bild.
